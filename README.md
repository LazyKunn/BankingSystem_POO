# Sistema Bancário POO

## Enunciado do Projeto 
Pretende-se desenvolver, em Java (Programação Orientada a Objetos), uma 
aplicação que simule o funcionamento de um sistema bancário interno com 
vários canais de acesso: ATM, WebBanking e Banco. Os canais devem partilhar a 
mesma lógica de negócio. A aplicação deve ser em consola, podendo 
opcionalmente incluir interface gráfica em JavaFX. 

## Objetivos de Aprendizagem 
O projeto deve promover a aplicação de princípios da Programação Orientada a 
Objetos, a separação entre lógica de negócio e canais de interação, a validação 
de operações bancárias e o trabalho colaborativo.
